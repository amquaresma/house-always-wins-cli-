# house-always-wins-cli-
House Always Wins – Educational Casino Simulator (CLI) é um programa educacional desenvolvido em Python, executado em modo console, com o objetivo de conscientizar usuários sobre o funcionamento real dos algoritmos utilizados em cassinos online.

O projeto reúne diferentes módulos interativos que simulam jogos de azar, testes de probabilidade e cenários estatísticos, permitindo ao usuário visualizar, na prática, como conceitos como RNG (Random Number Generator), vantagem da casa, probabilidade e expectativa matemática impactam os resultados a longo prazo.

Além das simulações, o sistema apresenta explicações educativas após cada interação, promovendo uma compreensão crítica sobre a ilusão de lucro, os mecanismos psicológicos envolvidos nas apostas e o motivo pelo qual, estatisticamente, a casa sempre ganha.

O projeto foi desenvolvido em Python puro, com arquitetura modular, foco em clareza de código e facilidade de depuração, sendo ideal tanto para fins educacionais quanto como demonstração de habilidades em lógica, estruturação de software e pensamento analítico.
